# Brand Guidelines — Steal Therapy

## Purpose
Defines the brand identity, tone of voice, and visual style for Steal Therapy.

## Brand Essence

### Tagline
**"Steal Forges Steel"**

### Mission
Helping individuals build physical resilience through structured fitness programming, with a focus on rehabilitation and long-term health.

### Core Values
1. **Discipline Over Motivation** — Consistency beats intensity
2. **Progress Over Perfection** — Small wins compound
3. **Strength Through Struggle** — Challenges forge character
4. **Own Your Journey** — No excuses, just results

## Visual Identity

### Color Palette
```css
/* Primary — Industrial Dark */
--brand-dark: #0a0a0a
--brand-charcoal: #171717
--brand-gray: #262626

/* Accent — Steel White */
--brand-white: #fafafa
--brand-silver: #a3a3a3

/* Functional Accents */
--brand-primary: #ffffff      /* Primary actions */
--brand-success: #22c55e      /* Progress, completion */
--brand-warning: #f59e0b      /* Alerts, cautions */
--brand-danger: #ef4444       /* Errors, destructive */
```

### Typography
- **Primary Font**: Inter (or system-ui fallback)
- **Weights**: 400 (regular), 600 (semibold), 700 (bold)
- **Scales**:
  - Display: 48px / 700 / -0.02em tracking
  - H1: 32px / 700 / -0.01em
  - H2: 24px / 600 / -0.01em
  - H3: 20px / 600 / 0
  - Body: 16px / 400 / 1.5 line-height
  - Small: 14px / 400 / 1.5 line-height

### Logo Usage
- Primary: Wordmark "STEAL THERAPY" in bold, uppercase
- Secondary: Icon + wordmark combo
- Always use on dark backgrounds for maximum contrast
- Minimum clearspace: equal to cap height of "S"

## Tone of Voice

### General Principles
- **Direct** — Say what you mean, no fluff
- **Encouraging but not coddling** — Supportive without being soft
- **Action-oriented** — Verbs over adjectives
- **Authentic** — No corporate speak, no hype

### Do's and Don'ts

| Do | Don't |
|----|-------|
| "Show up. Do the work." | "We're here to support your wellness journey!" |
| "Progress takes consistency." | "Remember, every step counts!" |
| "You've got this." | "Believe in yourself!" |
| "Log your session." | "Don't forget to capture your workout!" |

### Microcopy Guidelines

#### Buttons
- Primary: "Start Session", "Complete", "Save Plan"
- Secondary: "Skip", "Later", "Cancel"
- Avoid: "Submit", "Click Here", "Proceed"

#### Empty States
- "No sessions yet. Your first workout awaits."
- "No plans created. Build your foundation."
- Avoid generic "Nothing here yet"

#### Error Messages
- "That didn't work. Check your connection and try again."
- "Session save failed. Your data is safe—retry when ready."
- Avoid technical jargon, be human

#### Success States
- "Session logged. Steel forged."
- "Plan saved. You're building something."
- Avoid excessive celebration, keep it grounded

## UI Conventions

### Imagery
- Real people, real effort (no stock fitness models)
- High contrast, desaturated tones
- Focus on action, not posing

### Icons
- Lucide React (consistent stroke weight)
- 24px default size
- Use sparingly — text first, icons for clarity

### Animations
- Subtle transitions (150-200ms)
- No bouncing, no bouncing, no playful effects
- Progress indicators should feel weighty, not floaty

## Application

### Where These Guidelines Apply
- All UI copy and labels
- Email communications
- Push notifications
- Marketing materials
- Social media posts

### When in Doubt
Ask: "Does this feel like steel being forged?" If it feels soft, generic, or corporate, revise.
